# EduPulse

EduPulse is a dual-sided educational event management platform designed to simplify communication, event participation, and feedback collection between organizers and learners.

The platform helps educational institutions, coordinators, and organizations manage programs more efficiently while providing learners with a smooth and engaging experience.

---

# Features

## Learner Features

* Personalized event and program feed
* One-tap event registration
* Calendar integration for registered events
* Join sessions directly through the app
* Submit feedback after session completion
* Receive important announcements and updates

## Admin Features

* Create and manage educational events
* Share announcements and updates
* Manually manage attendance records
* Monitor participant engagement
* Receive feedback directly through email
* Access analytics and reports

---

# Problem Statement

Educational programs and workshops often face communication gaps, inefficient attendance systems, and delayed feedback collection.

EduPulse solves these issues by providing a centralized platform for:

* Event management
* Learner engagement
* Communication
* Feedback collection
* Attendance monitoring

---

# Core Functionalities

* Event discovery and registration
* Role-based authentication (Learner/Admin)
* Program listings with filters
* Session joining system
* Feedback submission system
* Analytics dashboard
* Announcement management

---

# User Flow

## Learner Flow

1. Login or create an account
2. Browse available programs
3. Register for events
4. Join sessions through the app
5. Submit feedback after the session

## Admin Flow

1. Create and publish events
2. Send announcements
3. Manage attendance manually
4. Monitor participation
5. Review feedback received through email
